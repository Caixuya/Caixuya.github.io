1. 什么是generator

列表生成式，我们可以直接创建一个列表。但是，受到内存限制，列表容量肯定是有限的，如果创建一个包含100万个元素的列表，不仅占用很大的存储空间，如果我们仅仅需要访问前面几个元素，那后面绝大多数元素占用的空间都白白浪费了。

如果可以先不创建完整的list，而是先创建一个东西，使得这个东西可以在循环的过程中计算出后面的元素，这样可以节省大量的空间。这个一边循环一边计算的机制，成为生成器：generator

三：可变参数

可变参数就是传入的参数个数是可变的，可以是1个、2个到任意个，还可以是0个

<1>主函数—没有元素、单个元素或者多个元素传入
def calc(*numbers):        #这里的*表示把number改成可变参数，让函数传入的参数个数是可变的。
    sum = 0
    for number in numbers:
        sum = sum + number * number        #这里是number乘number
    print(sum)
    return sum


def main():
    calc(1, 2, 3)
    calc(2, 3)
    calc()


if __name__ == "__main__":
    main()

输出结果为：

14
13
0

如果不把calc函数的参数变成可变参数，那么在主函数中调用的时候就要先组装出一个list或者tuple，如下所示

def calc(numbers):
    sum = 0
    for number in numbers:
        sum = sum + number * number
    print(sum)
    return sum


def main():
    calc([1, 2, 3])        #或calc((1, 2, 3))


if __name__ == "__main__":
    main()

输出结果为：

14

递归函数—递归，尾递归

函数在内部可以调用其他函数，也可以自己调用自己，自己调用自己的函数叫递归函数。

例子一：

如：

python
def fact(n):
    if n==1:
        return 1        
     #n * fact(n-1)中，当n=1时(n-1)=0，为了避免n=1时出错，直接让n=1时返回1
    return n * fact(n-1)

假设这里n=5,即 def fact(5)：

这个函数的执行过程是：先调用fact函数的返回值：n * fact(n-1)

fact(5) ==>n fact(n-1)，==> 5 fact(5 - 1) ==> 5 * fact(4)

这里fact（4）再次调用fact函数，即将算式中的fact(4)替换成返回值

fact(4) ==>n fact(n-1)，==>5 (4 fact(4 - 1)) ==> 5 (4 * fact(3))

以此类推，直到算式中没有fact()函数

fact(3) ==>n fact(n-1)，==>5 (4 (3 fact(3-1))) ==> 5 (4 (3 * fact(2)))

fact(2) ==>n fact(n-1)，==>5 (4 (3 (2 fact(2-1)))) ==> 5 (4 (3 (2 * fact(1))))

fact(1) ==>n fact(n-1)，==>5 (4 (3 (2 (2-1)))) ==> 5 (4 (3 (2 * fact(1))))

n = 1的是时，return 1 ，所以最终是：5 (4 fact(3 fact(2 1)))

因此fact(1) ==>n fact(n-1)，==>5 (4 (3 (2 (2-1)))) ==> 5 (4 (3 (2 * 1)))

即 5 4 3 2 1 = 5!=120

尾递归

def fact(n):
    print(fact_iter(n, 1))
    return fact_iter(n, 1)


def fact_iter(num, product):
    if num == 1:
        return product                
    #在这里打印可以看到递归（循环）的次数
    return fact_iter(num - 1, num * product)


if __name__ == "__main__":
    fact(3)

题1 确定合适的类型

属性名 特征 类型（宽度）
员工编号 4位数字。如1111 NUMBER(4)
员工姓名1 英文字母，不超过10个字母。
要尽量高的存储空间利用率		 VARCHAR2(10)
员工姓名2 汉字，不超过10个汉字
要尽量高的存储空间利用率		 NVARCHAR2(10)
岗位1 英文字母，不超过10个字母 VARCHAR2(10)
岗位2 汉字，不超过10个汉字 NVARCHAR2(10)
上级经理编号 4位数字 NUMBER(4)
入职日期 年-月-日 DATE
薪水 数字，最大5位数 NUMBE(5)
岗位津贴 数字，最大5位数 NUMBER(5)
所属部门编号 2位数字。如01 NUMBER(2)
证件照 图片 BLOB
自我介绍 视频 BLOB
简历 可能超过2000汉字 CLOB(2000)

题2 约束的种类

约束 关键字
主键约束 primary key
外键约束 foreign key
非空约束 not null
唯一值约束 unique
默认约束 default ‘默认值’
Check约束 举例：check(gender='男' or gender='女');
