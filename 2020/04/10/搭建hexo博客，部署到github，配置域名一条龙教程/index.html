搭建hexo博客，部署到github，配置域名一条龙教程

2020-04-10
Word count: 393   |   Reading time≈ 1 min

我们直接开始教程吧

  1. 在浏览器地址栏输入这个链接，注册github账户邮箱，注册前先看完第二条

    github.com

  2. 如果不想买域名的话，博客搭建好以后直接部署到github就行，这里要注意，部署到github的地址名字要和你的github的用户名一致，例如：github的用户名为Xiaoxiao,部署到github后，博客地址就是这个：xiaoxiao.github.io，所以github的用户名要慎重再慎重，我之前在这里踩了坑，弄了好久才弄好

  3. 安装nodejs

    前面的是centos系列的安装命令，后面的是ubuntu和deeping系列的安装命令，后面就只展示centos的啦，ubuntu和deeping的只需要把yum变成apt-get即可

    sudo yum install nodejs或者sudo apt-get install nodejs

  4. 安装npm

    sudo yum install npm

  5. 检查安装情况：分别输入以下命令，如果出现版本号则表示安装成功

    node -v
    npm -v

  6. 创建一个文件夹用来放博客

    mkdir blog

mkdir是Linux的创建文件的命令，命令后面跟着的就是文件的名字，名字个人喜欢命名
7. 进入到刚刚创建的文件夹中

cd blog
8. 安装hexo
很多人在安装npm的时候会习惯换成淘宝镜像的cnpm，因为npm的服务器位于国外可能会影响安装，我也是这样，但是安装hexo似乎不能直接用cnpm安装，emmm，什么原因我也不清楚，没去查过，所以我们不换成cnpm了，用npm安装，命令如下：
npm install -g hexo-cli
