    种类：+ - * /

  • 比较操作符
    功能: 对两个表达式的值进行比较

    | 种类 | 说明 |
    | :——: | :——: |
    | = | 等于 |
    | != | 不等于 |
    | > | 大于 |
    | < | 小于 |
    | >= | 大于等于 |
    | <= | 小于等于 |
    | IN(值1,值2,…) | 与列表中的值匹配 |
    | LIKE | 字符匹配模式。匹配符 %：任意个字符。 匹配符 ?: 一个字符 |
    | IS NULL | 判断字段值为空 |

    | 运算符 | 用法 |

    | NOT | Where not 条件 |

题1 确定合适的类型

属性名 特征 类型（宽度）
员工编号 4位数字。如1111 NUMBER(4)
员工姓名1 英文字母，不超过10个字母。
要尽量高的存储空间利用率		 VARCHAR2(10)
员工姓名2 汉字，不超过10个汉字
要尽量高的存储空间利用率		 NVARCHAR2(10)
岗位1 英文字母，不超过10个字母 VARCHAR2(10)
岗位2 汉字，不超过10个汉字 NVARCHAR2(10)
上级经理编号 4位数字 NUMBER(4)
入职日期 年-月-日 DATE
薪水 数字，最大5位数 NUMBE(5)
岗位津贴 数字，最大5位数 NUMBER(5)
所属部门编号 2位数字。如01 NUMBER(2)
证件照 图片 BLOB
自我介绍 视频 BLOB
简历 可能超过2000汉字 CLOB(2000)

题2 约束的种类

约束 关键字
主键约束 primary key
外键约束 foreign key
非空约束 not null
唯一值约束 unique
默认约束 default ‘默认值’
Check约束 举例：check(gender='男' or gender='女');
