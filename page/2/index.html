mysql ERROR 1045 (28000) Access denied for user 'ODBC'@'localhost' (using password NO)

Posted on 2020-05-01 Edited on 2022-04-14 In mysql
764 1 mins.

在cmd中敲入mysql后，出现下面的错误，找了很久没有找到解决方案

mysql ERROR 1045 (28000) Access denied for user ‘ODBC‘@’localhost’ (using password NO)

在找答案的过程中受一位网友的启发，翻译了这句话，尝试了一下，惊喜地发现困扰了我好几天的问题解决了

不过好像这个问题解决和不解决都没什么关系，我只是单纯强迫症犯了和这个问题刚起来了

这句话翻译过来就是你的mysql没有一个不用输入密码就可以登录的ODBC用户，解决方案就是：创建一个没有密码的ODBC用户，如下所示：

CREATE USER 'ODBC'@'localhost' IDENTIFIED BY '';
然后更新一下权限
flush privileges;

说到创建用户，版本不同命令是有些微差别的，如果上面这条命令不适用你的mysql，就去找找其他的创建用户的命令，这个问题的解决方案就到这里啦～～
附上图片：

另外附上mysql登录命令：mysql -u root -p;
关闭命令：service mysqld stop;
修改密码命令：set password for 'root'@'localhost'=password('991007');
赋予所有权限：grant all privileges on *.* to 'root'@'%' identified by 'root';
更新权限命令update mysql.user set Grant_priv='Y',Super_priv='Y' where user = 'root' and host = '%';

解决liux系统只读文件系统，linux系统磁盘上锁问题

Posted on 2020-05-01 Edited on 2022-04-14 In linux
674 1 mins.

装双系统的朋友可能经常会遇到这样的问题，某天打开linux系统的时候发现windows系统的所有磁盘都锁住了，文件移不出来，也没办法创建文件

之前找资料说重启可以解决，第一次重启的时候并没有解决，然后，我就重启了第二次，emmm，惊讶的发现居然可以了

而后很久都没有再出现这个问题，正当我觉得那次磁盘被锁只是意外时，某个夜高风黑的晚上，我打开linux系统，发现又锁了，我当时差点就问候我亲爱的电脑了，当然我忍住了

上网找解决方案的时候，这次终于皇天不负有心人，我找到了解决方案，我得抓紧这颗稻草，于是我连忙写在自己博客，防止链接丢失或者某天这个回答突然不见了

解决方案很简单，win + R打开运行（命令提示符），输入下面这行东东

  • df -h

然后可以看到你的磁盘的路径，然后再输入下面这行东东

  • sudo mount -o rw,remount /media/tia/系统

/media/tia/系统介个是我那个被锁住的磁盘的路径，在第一条命令输入后可以找到，rw后面是英文状态下的逗号，原回答有更详细的说明，下面会贴出原回答，我这里只贴出我尝试过的解决方案。

值得一提的是：ro为只读，rw为可读可写

（其实我认为会找这个的问题的解决方案的人大部分是因为想要解锁文件，所以并不会想知道文件只读的方法是啥，例如我就是这么想的，但是为了防止我以后要用到却找不到，所以我还是写下来吧）

如果发现有提示“device is busy”

  • fuser -m /boot 将会显示使用这个模块的pid
  • fuser -mk /boot 将会直接kill那个pid

端口占用

hexo默认的端口是4000，不确定是否被占用的情况下，打开终端输入yum install lsof安装lsof，其他系统的安装命令自查，然后输入命令lsof -i:4000查看

欢迎使用 Markdown在线编辑器 MdEditor

Markdown是一种轻量级的「标记语言」

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面，Markdown文件的后缀名便是“.md”

MdEditor是一个在线编辑Markdown文档的编辑器

MdEditor扩展了Markdown的功能（如表格、脚注、内嵌HTML等等），以使让Markdown转换成更多的格式，和更丰富的展示效果，这些功能原初的Markdown尚不具备。

Markdown增强版中比较有名的有Markdown Extra、MultiMarkdown、 Maruku等。这些衍生版本要么基于工具，如Pandoc，Pandao；要么基于网站，如GitHub和Wikipedia，在语法上基本兼容，但在一些语法和渲染效果上有改动。

MdEditor源于Pandao的JavaScript开源项目，开源地址Editor.md，并在MIT开源协议的许可范围内进行了优化，以适应广大用户群体的需求。向优秀的markdown开源编辑器原作者Pandao致敬。
